## Neighborhoods Problem Statement: 
Given an undirected graph, assign a direction to every edge to minimize the total number of Strongly Connected Components (SCCs). Output both the minimum SCC count and the resulting edge orientations.

## Solution $O((n + m)\ \log{m})$:
The solution is based on the graph-theoretic principle that the minimum number of Strongly Connected Components (SCCs) is determined by the number of bridges in the original undirected graph.

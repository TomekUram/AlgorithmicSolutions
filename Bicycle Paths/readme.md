## Problem Statement: Bicycle Paths (XXV OI)

[cite_start]For every vertex $v$ in a directed graph, calculate the number of reachable vertices[cite: 13], given the **"fairness" constraint**:

> [cite_start]If there is no path from $v$ to $u$, then there is at most one path from $u$ to $v$[cite: 8].

---

### Key Details
* [cite_start]**Source:** XXV Polish Olympiad in Informatics (OI), Stage II.
* [cite_start]**Task:** Compute reachability for each intersection in a fair network[cite: 12, 13].
* [cite_start]**Resource Constraints:** 64 MB memory limit.
